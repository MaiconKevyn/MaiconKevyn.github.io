# Public Assets

This directory contains the static assets for the Vercel portfolio.

## Profile Image

- `profile.png` is the main profile image used by the hero.
- `profile.webp` is an optional alternative asset kept for future optimization work.

## Project Cards

The `images/` directory contains project artwork used by the featured-project carousel.
These can be replaced with real screenshots later as long as the file paths referenced by `src/components/Projects.tsx` stay in sync.

# MaiconKevyn.github.io

Static portfolio site for GitHub Pages.

## Local preview

Because this is a plain HTML, CSS, and JavaScript site, you can preview it with any local static server.

Example:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.

## Publish

Push the contents of this repository to the `main` branch of `MaiconKevyn.github.io`.

If GitHub Pages does not publish automatically, set:

- `Settings` -> `Pages`
- `Build and deployment` -> `Deploy from a branch`
- `Branch` -> `main`
- `Folder` -> `/(root)`

The site URL should be:

`https://maiconkevyn.github.io/`
